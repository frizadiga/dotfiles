# friz()'s dotfiles
My daily driver public dotfiles.

- `alacritty`  
  terminal emulator
- `alttab`  
  window level switcher
- `apt`  
  debian package manager
- `btop`  
  terminal resource monitor
- `brew`  
  system package manager
- `cmder`  
  terminal emulator for Windows
- `ghostty`  
  terminal emulator
- `hammerspoon`  
  automation tool
- `karabiner`  
  key remapper
- `lazydocker`  
  docker terminal UI
- `lazygit`  
  git terminal UI
- `opencode`  
  AI code assistant
- `powertoys`  
  Windows system utilities
- `scoop`  
  Windows package manager
- `skhd`  
  hotkey daemon
- `tmux`  
  terminal multiplexer
- `v`  
  highly customizable text editor that sucks less
- `winget`  
  Windows package manager
- `yabai`  
  window manager
- `yazi`  
  terminal file manager

All symlinking mechanisms are handled by [my internal tooling](https://github.com/frizadiga/symlinker). If you want to reproduce this setup, you'll need to provide your own symlinking solution.
